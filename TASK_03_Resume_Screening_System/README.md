# Resume Screening System

## Project Overview

This project implements an AI-based Resume Screening System using Natural Language Processing (NLP).

The system automatically compares resumes with a job description and ranks candidates according to their similarity score.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- TF-IDF
- Cosine Similarity
- Matplotlib
- Jupyter Notebook

## Workflow

1. Load dataset
2. Explore dataset
3. Clean resume text
4. Remove duplicates
5. Convert resumes into TF-IDF vectors
6. Transform job description
7. Calculate cosine similarity
8. Rank resumes
9. Visualize top candidates

## Results

The system successfully identified the resumes most relevant to a Data Scientist position using cosine similarity.

Top candidate similarity score:

41.48%
